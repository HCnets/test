# 页面设计文档（桌面优先 + 兼容竖屏）

## 全局设计
### Layout
- 桌面优先：以 1200px 内容宽为基准（居中）；小屏降级为单列竖排。
- 布局方式：主区域用 CSS Grid（两列：章节树/详情），移动端折叠为 Tab（章节树/详情）。
- 交互控件保持大点击热区（≥44px），保证抖音小游戏触控友好。

### Meta Information（通用）
- title：作品名 + 当前章节/场景（例如《XX》- Chapter 2）
- description：一句话简介 + 当前进度提示
- Open Graph：使用本地默认封面图（若运行环境不支持OG则忽略）

### Global Styles（Design Tokens）
- 背景：#0B0F1A（深夜蓝黑）
- 主色：#7C5CFF（霓虹紫）
- 辅色：#2DE2E6（霓虹青）
- 强调/危险：#FF3D81
- 字体：系统字体栈（优先苹方/思源黑体同等系统替代）
- 文本层级：
  - H1 28/32
  - H2 20/28
  - Body 16/24
  - Caption 12/18
- 按钮：圆角 12，默认霓虹描边；hover/press 有亮度+阴影变化（移动端以press为主）
- 卡片：半透明磨砂（rgba(255,255,255,0.06)）+ 轻边框（rgba(255,255,255,0.12)）

## 页面 1：首页/章节树
### Page Structure
- 顶部：标题栏（作品名、当前存档槽、设置入口）
- 主体：左侧章节树图/列表，右侧章节详情卡（简介、解锁条件、入口按钮）
- 底部：继续游戏、收藏入口、快速存档提示

### Sections & Components
1) 顶栏（TopBar）
- 左：作品名 + 版本号（开发者模式可见）
- 中：当前进度（Chapter/Ending收集率）
- 右：设置按钮

2) 章节树（ChapterGraph）
- 视图：
  - 桌面：可缩放/拖拽的节点图（章节→关键节点→结局）
  - 移动：折叠为纵向卡片时间线 + “展开图”按钮
- 节点状态：
  - 已解锁：发光边框
  - 未解锁：灰置 + 解锁条件tooltip
  - 隐藏：显示“？？？”节点，条件满足后显形
- 点击节点：右侧详情同步更新

3) 章节详情（ChapterDetailCard）
- 信息：章节标题、时长预估、已读率、分支数、结局数
- 行为：开始/继续、选择存档槽、查看可回溯点分布（小图标）

4) 继续/收藏快捷（QuickActions）
- 主按钮：继续上次
- 次按钮：进入收藏与制作

## 页面 2：剧情播放页
### Page Structure
- 全屏沉浸：背景层（BG）+ 角色层（CHAR）+ 特效层（FX）+ UI层（TextBox/Choices/Controls）
- 桌面：右侧可选信息栏（变量/已读/节点ID，仅开发者模式）

### Sections & Components
1) 画面舞台（Stage）
- 背景图：支持淡入淡出、轻微视差（可选）
- 立绘：左/中/右三锚点；支持表情切换
- 特效：屏闪、噪点、霓虹扫描线（可配置强度）

2) 文本框（TextBox）
- 包含：说话人、正文、跳过/下一句
- 支持：逐字打印、点击加速、长按全显
- 已读标记：已读句子显示轻微“已读”点缀

3) 选项列表（ChoiceList）
- 样式：竖排卡片按钮；条件不满足时灰置并显示原因
- 隐藏选项：默认不占位；开发者模式可显示占位与条件

4) 控制条（PlayerControls）
- 回溯（Rollback）：返回上一个可回溯点
- 自动：自动播放开关
- 快进：仅已读快进
- 菜单：打开设置/存档/退出到章节树

5) 存档弹窗（SaveModal）
- 多槽：至少3槽 + 自动存档
- 信息：截图缩略（可选）、章节名、时间、关键变量摘要

6) 结算卡（EndingCard）
- 展示：结局名、获得标签、解锁提示、返回章节树

## 页面 3：收藏与制作（含开发者模式）
### Page Structure
- Tabs：结局图鉴 / 进度统计 /（开发者模式）配置与校验

### Sections & Components
1) 结局图鉴（EndingGallery）
- 布局：九宫格（桌面4列/移动3列）
- 卡片：结局名、解锁时间、回到对应章节入口
- 未解锁：显示剪影 + 条件提示（不剧透的版本）

2) 进度统计（ProgressPanel）
- 指标：章节完成度、分支探索度、隐藏内容发现率
- 入口：推荐你下一步去的章节（基于缺失结局/最近游玩）

3) 开发者模式（DevToolsPanel）
- 配置模板说明：字段解释 + 示例（仅文档/说明，不做外链）
- 导入/校验：
  - 导入（选择本地文件/或粘贴JSON文本，取决于运行环境能力）
  - 校验结果：错误列表（可定位到sheet/行号/字段）
- 资源规范清单：目录/命名/格式/预算（可一键复制）
- 自动化测试：展示脚本列表与最近一次报告（report.md摘要）

## 响应式与动效建议
- 断点：≥1200（桌面双列）、768~1199（窄桌面双列缩紧）、<768（移动单列/Tab）。
- 动效：转场 120~220ms；按钮按压 80~120ms；避免大面积模糊导致性能问题。
- 可访问性：高对比模式开关（可放在设置页）。
