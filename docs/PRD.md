# 23:59 的赛博抽屉（小程序版）内容与玩法深度提升 PRD（v0.1）

## 1. 背景与目标
- 当前形态：抽屉式入口 + 视觉小说（VN2）+ 解谜关卡（镜面建筑）+ 任务/成就/背包/快捷栏抽屉
- 新方向：解谜微恐（克制、不跳吓）+ 当代年轻人情绪叙事（疲惫、意义感、边界、自我价值）

## 2. 核心指标（可本地采样）
- 次日留存：+8%（灰度 A/B）
- 7日留存：+5%（灰度 A/B）
- 关键完成率：解谜关卡通关率 ≥60%
- QTE 成功率：60%~85%（动态调整）
- 崩溃：连续30分钟无崩溃日志（本地报告 + QA复现）

## 3. 内容策略
### 3.1 主线（≥8幕）
- 每幕 3~4 个选项，影响：
  - NPC 好感（affinity）
  - Flags（解锁/隐藏/支线前置）
  - 任务奖励（资源）
  - 微恐演出（bubble/flash/shake）

### 3.2 支线任务（≥15条，任务网）
- 结构：前置条件 → 进行中 → 可领取 → 后置连锁解锁
- 机制：时间窗（morning/afternoon/night）+ 玩法触发（解谜/抽屉回合/VN选择）

### 3.3 隐藏彩蛋
- 夜晚限定选项、特定好感阈值触发、特定解谜通关方式触发

## 4. 玩法策略
### 4.1 解谜微恐（核心）
- 通过“视觉错位 + 旋转/升降/颜色门”形成可学习的解谜语言
- 用克制的视听（暗角、轻噪点、耳语、低频震动）建立情绪张力

### 4.2 每日不重样（轻 Roguelike）
- 方案：每日种子生成「房间链」= 叙事节点 + 解谜节点 + 奖励节点
- 每天给出1次“每日跑图”，奖励用于养成线（资源/称号/好感）

### 4.3 异步排行 & 轻社交（小程序可落地形态）
- 排行：以「每日跑图分数」上榜（后端版：云存储；离线版：本机榜/分享码）
- 好友助战：分享“助战码”= 一次性增益（降低QTE门槛/额外提示次数）
- 公会捐赠：后端版；离线版用“自我合约”替代（每天一次捐赠换奖励）

## 5. 经济系统（双货币+稀缺资源）
- 软货币：碎片（红豆/糖/蹄印）= 日常产出、用于合成/养成/快捷道具
- 硬货币：镜币（待接入）= 活动/付费点（后端版）
- 稀缺资源：墨水/热度/专注（Run内）= 决策权与策略深度

## 6. 动态难度（DDA）
- QTE：根据最近 N 次成功率动态调整窗口（650~900ms）
- 解谜提示：根据失败次数/提示点击率调整“提示强度/高亮”

## 7. 灰度与A/B
- A组：当前默认节奏
- B组：更强微恐演出 + 更严格QTE（但有补偿奖励）
