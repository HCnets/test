# 数值与经济系统（v0.1）

## 1) 货币与资源
| 类型 | 名称 | 产出 | 消耗 | 设计目的 |
|---|---|---|---|---|
| 软货币 | 红豆 | B抽屉选择/任务领取 | 合成/换道具 | 情绪回礼/关系推进 |
| 软货币 | 糖 | A抽屉/任务领取 | 快捷治愈/稳定心情 | 自我修复/稳定器 |
| 软货币 | 蹄印 | C抽屉/任务领取 | 解锁跑图节点/加速 | 行动证据/推进力 |
| 稀缺资源 | 墨水 | 开局/任务 | B“加印寄出”等 | 抉择权/收益换风险 |
| 稀缺资源 | 热度 | A特殊机制/任务 | 触发高回报 | 高波动玩法阀门 |
| 稀缺资源 | 专注 | C漂移/任务 | 提升成功率 | 技巧补偿 |

## 2) 资源循环（防通胀）
- 日常产出主要来自：任务链、每日跑图、解谜通关
- 大额消耗来自：合成/升级/解锁（长期 sink）
- 稀缺资源只在 Run 内循环，避免长期累积导致通胀

## 3) 付费点（后端版规划）
- 镜币（硬货币）用于：皮肤/主题/活动通行证/加速（不卖强度核心）
- 付费获得“表达权”而非“通关权”：更多剧情分支皮肤、演出包、限定支线
