# 战斗系统（连击 + 元素反应）v0.1

## 1) 入口
- 工位（Hub）→「每日/每周挑战（战斗+掉落）」→ 开始

## 2) 核心循环
- 选元素 → 连续攻击维持连击 → 触发元素反应 → 击败敌人 → 掉落装备/材料 → 进入下一战/结算总分

## 3) 元素（10种）
- 火/水/冰/电/风/土/金/木/光/暗

## 4) 反应（示例 ≥30 组合里已先落地一批）
- 火+水：蒸汽（伤害提升）
- 火+冰：融化（高伤害）
- 电+水：感电（伤害+持续）
- 电+冰：超导（伤害+易碎）
- 火+电：过载（高伤害）
- 风+任意：扩散（扩散伤害）
- 暗+光：蚀（高伤害）

## 5) 天气/昼夜
- 按系统时间分清晨/下午/夜晚
- 天气：晴/雨/雾/大风/雷暴（影响元素倍率）

## 6) 成长联动
- 挑战胜利：+1 技能点
- 装备：稀有度分层 + 随机词条
- 技能树：影响连击窗口、护盾、伤害、分数等

## 7) 数据采集
- battle_start / battle_attack / battle_hit / battle_end
- item_gain / skill_unlock
